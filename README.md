# 🚍 Bus Reservation System

A simple **Console-Based Java Application** that allows users to view buses, book seats, and view booking records.

This project demonstrates core **Object-Oriented Programming (OOP)** concepts using Java.

---

## 📌 Project Type

* Language: Java
* Interface: Console (Command Line)
* Database: ❌ Not Used
* Framework: ❌ None
* IDE: Any (VS Code / Eclipse / IntelliJ)

---

## 📂 Project Structure

```
Bus-Reservation-System/
│
├── Bus.java
├── Booking.java
├── BusDemo.java
└── README.md
```

---

## 🧠 Features

✔ View available buses
✔ Book seat for a passenger
✔ Validate seat availability
✔ View all booking records
✔ Menu-driven interface

---

## 🏗️ OOP Concepts Used

* Encapsulation
* Classes and Objects
* Association
* ArrayList (Collections Framework)
* Conditional Logic
* Looping (Menu System)

---

## 📊 Class Overview

### 1️⃣ Bus

Handles:

* Bus details
* Seat availability
* Seat booking

### 2️⃣ Booking

Handles:

* Passenger details
* Bus number
* Seat number

### 3️⃣ BusDemo

* Main class
* Controls application flow
* Displays menu
* Manages bookings

---

## ▶️ How to Run

### Step 1: Compile

```bash
javac Bus.java Booking.java BusDemo.java
```

### Step 2: Run

```bash
java BusDemo
```
